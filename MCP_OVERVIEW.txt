WHAT IS THIS?
-------------
A Model Context Protocol (MCP) server that allows AI assistants like Claude
to generate memes programmatically by connecting to your local Memegen instance.

WHAT'S INCLUDED?
----------------
Core Files:
  - mcp_server.py              Main MCP server implementation
  - mcp_requirements.txt       Python dependencies for MCP server
  - mcp_server_config.json     Example configuration for MCP clients

Setup & Management Scripts:
  - install_mcp.bat           Install MCP dependencies
  - start_mcp.bat             Start the MCP server
  - test_mcp.bat              Test MCP server functionality
  - publish_mcp.bat           Publish to PyPI/npm/GitHub

Documentation:
  - MCP_README.md             Full documentation
  - MCP_QUICK_START.md        5-minute setup guide
  - MCP_OVERVIEW.txt          This file

Distribution Files:
  - setup_mcp.py              PyPI package setup
  - package_mcp.json          npm package configuration
  - MANIFEST_MCP.in           Distribution manifest

Test Suite:
  - test_mcp.py               Automated test suite

QUICK START COMMANDS

AVAILABLE TOOLS (5 Tools)
--------------------------
1. list_templates           - List all meme templates
2. get_template             - Get template details
3. generate_meme            - Create memes from templates
4. generate_custom_meme     - Create memes from custom images
5. search_meme_examples     - Search for meme examples

TROUBLESHOOTING
---------------
Issue: Cannot connect
Fix:   .\start.bat (start Memegen first)

Issue: Module not found
Fix:   .\install_mcp.bat (install dependencies)

Issue: Claude doesn't see tools
Fix:   - Check config path (use \\ on Windows)
       - Restart Claude Desktop
       - Verify with .\test_mcp.bat
